Sponza scene (TSF format)
-------------------------

Filename: sponza.tsf

This scene file is based on the .obj model released by Crytek (http://www.crytek.com/downloads/technology/).

The following changes have been made:
- Nodes containing multiple separate geometries have become separate nodes (e.g. the vases);
- Repeated geometry has been removed and replaced with instances;
- The floor has been split from the atrium mesh into a separate mesh;
- The vegetation has been removed from the vases;
- New materials have been assigned without using textures;
- The large hanging cloth has been removed.

Please give credit to the original authors when using this scene file.
